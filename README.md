# Simple Calculator

A small Python project that lets the user perform basic arithmetic operations interactively.  

## Features
- Addition (+), Subtraction (-), Multiplication (*), Division (/), and Modulus (%)
- Handles division by zero gracefully
- Interactive input for numbers and operation
- Optional loop for multiple calculations

## How to Run
1. Make sure Python 3.x is installed
2. Navigate to the project folder in terminal
3. Run the program:
python main.py

## Structure

simple_calculator/
├── main.py        # Main logic: input/output and calling functions
├── utils.py       # Functions for each operation
├── README.md      # Project description
└── .gitignore     # Ignore temp files like __pycache__
